Detailed Description

The EWPTReturnType struct Contains the following information.

Parameters
StatusFlag= SUCCESS : successful
StatusFlag= NOTVANISHINGATFINALTEMP: v(T=TempEnd) != 0 => vc = v(TempEnd) , Tc = TempEnd
StatusFlag= NOTNLOSTABLE: Not vacuum stable at T=0 (if TempStart = 0) or v(TempStart) = 0 or v(TempStart) > 255 => Tc = TempEnd, vc = 0
StatusFlag= NUMERICALLYUNSTABLE: v(T) > 255 during the bisection (this implies the parameter point is numerically unstable) => vc and Tc are the last VEVs encountered
StatusFlag= BELOWTHRESHOLD: v/T < C_PT during the bisection => vc = Last VEV, TC = Last Temp
vc= critical VEV
Tc= critical Temperature
